ACERCA DE MI

Soy un egresado de la carrera de ingeniería en computación. Me apaciona la tecnología, mis fortalezas son un gran gusto por aprender y la adaptabilidad al cambio de paradigmas. Algunos de mis hobbies son la pintura, videojuegos y la horticultura, actualmente busco profesionalizarme en una empresa de desarrollo de software ya que desde antes de egresar de la carrera e trabajado en proyectos y con personas cuyo enfoque siempre fue resolver sus problemas a través de la tecnología

Sistema punto de venta para gimnasios con PHP

El proyecto, como su nombre lo describe fue realizado para administrar las membresías y la venta de productos, como suplementos a los clientes del gimnasio "SACRO" ubicado en tequixquiac, Edo. de México.

Fue elaborado con HTML, Bootstrap, JavaScript, MySQL y PHP. Haciendo uso también de las herramientas fpdf y Composer.

Puedes echarle un vistazo al código en Github




Sistema de gestión empresarial desarrollado en C#

Aplicación de escritorio para windows. Desarrollada para llevar a cabo un control de egresos e ingresos respecto a gastos con proveedores e insumos además de contabilizar las utilidades de cada trabajo realizado.

Fue elaborada con el patrón de diseño MVC utilizando la tecnología de Windows forms en lenguaje C# Y para la base de datos se utilizó SQL Server. Haciendo uso también de las herramienta de Microsoft REPORT VIEWER para generear reportes en excel y pdf.

Puedes echarle un vistazo al código en su respectivo repositorio en Github
